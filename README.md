## 🧑‍💻 Programming Languages (Note this is BriskLabs Stack)
- **[Swift](https://swift.org)** – Apple's language for building apps on iOS, macOS, and beyond  
- **[Kotlin](https://kotlinlang.org)** – Modern language for Android and cross-platform development  
- **[Dart](https://dart.dev)** – Used with Flutter for building cross-platform mobile, web, and desktop apps  
- **[JavaScript / TypeScript](https://www.typescriptlang.org)** – Ubiquitous language for frontend, backend (Node.js), and scripting; TypeScript adds static typing for scalability  
- **[Python](https://www.python.org)** – Versatile scripting language popular in automation, web APIs, and machine learning  
- **[Go (Golang)](https://go.dev)** – Fast, statically typed language great for backend services and networking  
- **[Rust](https://www.rust-lang.org)** – High-performance, memory-safe systems language used in backend and embedded development  
- **[C](https://en.cppreference.com/w/c)** – Low-level language for operating systems, firmware, and embedded systems  
- **[C++](https://isocpp.org)** – General-purpose language for high-performance applications like robotics and gaming  
- **[MicroPython](https://micropython.org)** – Python variant optimized for microcontrollers and IoT devices
